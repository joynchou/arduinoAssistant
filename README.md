# arduinoAssistent

随着越来越多的人加入到创客的队伍中，arduino的人气也是只增不减，自然就会有人没有基础知识，但是也想入门，这样一款软件就此诞生了，一款专门为初学者准备的教学app，并提供相关的查询功能

#### 产品功能

 • 可以在软件中帮助初学者学习各种模块及示例代码
 
 • 可以在软件中了解到各种所需知识
 
 • 查询各类arduino的硬件及硬件参数
 
 • 查询各种常用模块的主要参数
 
 • 查询各种官方api
 
 • 模块购买链接
 
 #### 实现阶段
 此软件现在还在开发阶段，并且现在只开发了硬件查询模块
